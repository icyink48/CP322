## Project Overview

This project predicts whether an individual's income is greater than 50K or less than or equal to 50K using demographic and employment-related attributes from the Adult Income dataset.

The goal is to build, train, compare, and evaluate multiple machine learning classification models, then determine which model performs best on the task.

## Dataset

This project uses the Adult Income dataset, which contains demographic and employment-related information such as:

- age
- workclass
- fnlwgt
- education
- education-num
- marital-status
- occupation
- relationship
- race
- sex
- capital-gain
