# Project Abstract

Predicting income level is an important machine learning problem because it helps identify relationships between demographic, educational, and employment-related factors and an individual’s earning category. This project addresses the problem of predicting whether a person earns more than $50,000 per year, which is formulated as a binary classification task. The main objective is to build and compare multiple supervised machine learning models while demonstrating proper data preprocessing, model evaluation, and hyperparameter tuning. The project also examines important machine learning issues such as missing values, encoding categorical variables, feature scaling, overfitting, and generalization to unseen data.

The dataset used in this study is the Adult (Census Income) dataset from the UCI Machine Learning Repository. It contains 48,842 records and 14 predictor variables, including age, education, occupation, marital status, hours worked per week, and native country. The target variable is income, with two classes: <=50K and >50K. The methodology begins with a baseline DummyClassifier, followed by Logistic Regression, Random Forest, and Linear Support Vector Machine (SVM) models. Performance will be evaluated using accuracy, precision, recall, F1-score, confusion matrix, and ROC-AUC where appropriate. The overall goal is to identify the best-performing model and achieve at least 80% classification accuracy.
